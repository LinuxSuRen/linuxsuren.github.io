大部分的缩写可以用于写作和口语，下面标注了“写作”的通常只用于写作：

| | | |
|---|---|---|
| ASAP | as soon as possible | |
| RSVP | please reply | |
| RIP | rest in peace | |
| BYOB | bring you own beer/booze | |
| BBQ | barbecue | |
| PIN | personal identity number | |
| e.g. | example | 写作 |
| i.e. | that is, restate an idea | |
| etc. | et cetera, and so forth | |
| ATM | automatic teller machine | |
