# Dependency

`go get -v github.com/zmb3/gogetdoc`
`go get -v golang.org/x/tools/cmd/guru1`
`go get -v github.com/golang/lint/golint`
`go get -v github.com/davidrjenni/reftools/cmd/fillstruct`
