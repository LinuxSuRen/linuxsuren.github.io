# Installation

`git clone https://github.com/tpope/vim-fugitive.git ~/.vim/bundle`

# Usage

`:Gstatus` 查看变更状态，等同于 `git status`

`:Gcommit %` 提交代码，等同于 `git commit`

`:Gpush` 推送代码

`:Gpush -u origin test` 推送新的分支

`:Gpull` 拉取代码
